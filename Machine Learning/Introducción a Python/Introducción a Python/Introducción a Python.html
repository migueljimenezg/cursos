Introducción a Python¶

Variables:¶

Las variables pueden tener un nombre corto como una sola letra, por ejemplo, x o pueden ser más descriptibos como nombre, estrato, peso, etc.

El nombre de la varible debe empezar por una letra y no por un número, puede tener guión medio -, guión bajo _, números y letras en mayúsculas: nombre_1, Nombre_1. Estas dos variables son diferentes porque la segunda tiene una mayúscula, igualmente, la variable peso es diferente a PESO.

nombre = "Juan"
nombre2 = "Jose"
peso = 60

print(nombre, nombre2, peso)

Juan Jose 60

Se puede usar las dobles comillas " " o las comillas sencillas ´´

nombre = "Juan"
nombre2 = "Jose"
print(nombre, nombre2)

Juan Jose

Se pueden asignar valor a varias variables en una sola línea:

nombre, nombre2, peso = "Juan", "Jose", 60
print(nombre, nombre2, peso)

Juan Jose 60

Si los valores están en una lista, se pueden asignar a las variables en una sola línea:

valores = ["Juan", "Jose", 60]
nombre, nombre2, peso = valores
print(nombre, nombre2, peso)

Juan Jose 60

Tipos de datos:¶

str: string.

int, float complex: numéricos.

list, tuple, range: secuencia.

bool: boleano.

Entre otros.

Para saber el tipo de dato se usa la función type().

nombre = "Juan"
type(nombre)

str

peso = 60
type(peso)

int

int son números enteros sin el punto decimal.

edad = 40.0
type(edad)

float

Note que los datos del tipo float tienen decimales, se expresan con números y el ., es común que para asignar el tipo de float solo es necesario el punto así:

edad = 40.0
type(edad)

float

x = list(("Juan", "Jose", 40))
print(x)
type(x)

['Juan', 'Jose', 40]

list

x = tuple(("Juan", "Jose", 40))
print(x)
type(x)

('Juan', 'Jose', 40)

tuple

x = range(8)
print(x)
type(x)

range(0, 8)

range

x = dict(nombre="Juan", edad=40)
print(x)
type(x)

{'nombre': 'Juan', 'edad': 40}

dict

Para especificar el tipo de dato se usan las siguientes funciones: int(), float(), str().

a = int(20)
b = float(4)  # con esta función no es necesario agregar el punto.
c = int("25")  # "25" es un string, pero esta función lo convierte en número.
d = str("Juan")
e = str(100)  # 100 es numérico pero esta función lo convierte en string
print(a, b, c, d, e)

20 4.0 25 Juan 100

type(e)

str

frase = "Esta es una frase en una línea"
print(frase)

Esta es una frase en una línea

Las tres comillas dobles """ o sencillas ''' permiten crear string en varias líneas así:

frase = """Esta es una frase muy larga
de más de una línea de
código"""
print(frase)

Esta es una frase muy larga
de más de una línea de
código

Operaciones:¶

+: Adición.

-: Sustracción.

*: Multiplicación.

/: División.

%: Módulo. Es el residuo de una división.

**: Exponencial.

//: División, pero aproxima al entero menor.

5 / 5

1.0

5 % 5

0

5 % 15

5

5 // 15

0

10 / 3

3.3333333333333335

10 // 3

3

==: Igual.

!=: Diferente.

>: Mayor que.

<: Menor que.

>=: Mayor e igual que.

<=: Menor e igual que.

&: AND.

|: OR.

3 == 3

True

3 == 5

False

3 + 5 & 7 + 1 == 8

True

Listas:¶

Con las listas se almacenan varios elementos en una sola variable. Similares a las listas están tuple (tuplas), set y dicy (diccionarios).

Las listas se crean usando corchetes [].

Los elementos de la lista tienen un orden de ubicación, es decir, están indexados, el primer elemento es el índice cero [0] (index 0), este es una gran diferencia entre otros lenguales de programación como R donde el primer elemento es el 1.

El segundo elemento de una lista tiene el index [1] y así sucesivamente.

lista = [542.0, 20, 20, 20, "Juan", "Jose", True]
print(lista)

[542.0, 20, 20, 20, 'Juan', 'Jose', True]

type(lista)

list

Para saber la cantidad de elementos de la lista se usa la función len().

len(lista)

7

Otra forma de crear la lista anterior es con la función list(). En este caso no se usan los corchetes [].

lista = list((542.0, 20, 20, 20, "Juan", "Jose", True))
print(lista)

[542.0, 20, 20, 20, 'Juan', 'Jose', True]

En resumen, las listas list son un conjunto de elementos ordenados modificables y permite duplicados. Tienen corchetes [].

Las tuplas tuple son un conjunto de elementos ordenados, inmutables y permite duplicados. Tienen paréntesis [].

set es un conjeto desordenado, inmutable, no indexado y no hay duplicados. Tinen corchetes {}.

dict es un conjunto ordenado, modificable y no hay elementos duplicados. Tinen corchetes {}, pero con clave y valor.

Tupla:

tupla = ("Juan", "Jose", 40)
print(tupla)

('Juan', 'Jose', 40)

type(tupla)

tuple

un_set = {"Juan", "Jose", 40}
print(un_set)

{40, 'Jose', 'Juan'}

type(un_set)

set

diccionario = {"clave": "valor"}
print(diccionario)

{'clave': 'valor'}

type(diccionario)

dict

diccionario_2 = {"Juan": 25, "Jose": 40, "Esteban": "Hombre"}
print(diccionario_2)

{'Juan': 25, 'Jose': 40, 'Esteban': 'Hombre'}
