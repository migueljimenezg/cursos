Taller Clustering Estados Financieros
-------------------------------------

Usar los cuatro archivos de los Estados Financieros de las empresas para
determinar una forma de agrupamiento utilizando los métodos de
clustering aprendidos. Emplear más de tres variables para agrupar las
empresas, pudiendo incluir más cuentas, como el Total de Activos, o
calcular ratios adicionales, como el ROA y el ROE.

Hacer lo anterior para una o varias industrias al mismo tiempo.
